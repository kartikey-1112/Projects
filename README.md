# 🚀 Multi-Project Portfolio: AI, Computer Vision & Web Dev

This repository is a curated collection of my independent projects, showcasing a diverse technical stack ranging from frontend web development to advanced machine learning and computer vision solutions.

---

## 📂 Project Showcase

### 1. 🧠 Smart Signature Verification
An AI-powered biometric authentication system designed to detect document forgery.
* **Tech Stack:** Python, Django, OpenCV, Tesseract, TensorFlow (CNN), SQLite3.
* **Key Feature:** Uses Transfer Learning and image pre-processing (noise removal/angular rotation) to verify signature authenticity with a probability match.
* **Deployment:** Heroku.

### 2. 😊 Emotion Detection System
A real-time computer vision application that analyzes facial expressions.
* **Tech Stack:** Python, OpenCV, TensorFlow/Keras, NumPy.
* **Key Feature:** Processes live webcam or image feeds to classify human emotions such as Happy, Sad, Angry, and Surprise using deep learning models.

### 3. 🧠 Memory Card Game
An interactive cognitive challenge built to test and improve memory skills.
* **Tech Stack:** HTML5, CSS3, JavaScript (DOM Manipulation).
* **Key Feature:** Implements complex game logic for card flipping, match detection, and automatic resets within a responsive, animated UI.

### 4. ✊✋✌️ Rock Paper Scissors
A classic, logic-driven web game played against an automated opponent.
* **Tech Stack:** HTML, CSS, JavaScript.
* **Key Feature:** Features a randomized choice-generation algorithm and instant result processing with real-time score tracking.
