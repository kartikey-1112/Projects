# 🧠 Memory Card Game

An interactive and engaging **Memory Card Matching Game** built using **HTML, CSS, and JavaScript**. The game challenges users to test and improve their memory skills by matching pairs of hidden cards in the least number of moves.

---

## 🚀 Features

- ✅ Flip cards to reveal hidden cars  
- ✅ Match pairs of identical cards  
- ✅ Cards flip back if they do not match  
- ✅ Interactive and responsive user interface  
- ✅ Move tracking / game logic (if implemented)  

---

## 🖥️ How It Works

1. Cards are placed face down, each containing a hidden cars  
2. The player flips two cards at a time  
3. If the cards match, they remain visible  
4. If they do not match, they flip back  
5. The game continues until all pairs are matched  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS  
- **Logic:** JavaScript (DOM Manipulation & Event Handling)
