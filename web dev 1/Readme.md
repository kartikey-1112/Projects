# ✊✋✌️ Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors game** built using **HTML, CSS, and JavaScript**. This project allows users to play against the computer in a fun and responsive web interface.

---

## 🚀 Features

- ✅ Play Rock, Paper, Scissors against the computer  
- ✅ Random computer choice generation  
- ✅ Instant result display (Win / Lose / Draw)  
- ✅ Score tracking (if implemented)  
- ✅ Interactive and user-friendly UI  

---

## 🖥️ How It Works

1. User selects Rock, Paper, or Scissors  
2. Computer randomly selects its move  
3. The game compares both choices  
4. Result is displayed instantly  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS  
- **Logic:** JavaScript
